Research研究方向

TL;DR. I build statistical methods that make small-sample, high-dimensional, multimodal scientific data behave — with provable identifiability and physical priors baked in. My current flagship lines are (i) doubly-debiased Lasso + GRF for cotton yield–climate attribution, and (ii) functional contrastive sparse multi-task learning on hyperspectral agricultural data.

一句话总结。我做的方法学针对小样本、高维、多模态 的科学数据，强调可识别性证明与物理先验嵌入。 目前两条主线：(i) 双重去偏 Lasso + GRF 做棉花产量–气候归因； (ii) 函数型对比稀疏多任务学习处理高光谱农产品数据。

Multi-basis representations (B-spline / wavelet / Fourier), functional contrastive PCA, semi-functional partial linear models, and function-on-function regression with mechanism priors.

Multimodal Sparse Learning

多模态稀疏学习

Heterogeneous-multimodal tensor-sparse fusion across meteorological / soil / remote-sensing / phenological streams; tri-modal image / spectrum / functional alignment for hyperspectral agricultural sensing.

异构–多模态张量稀疏融合：气象 / 土壤 / 遥感 / 物候四源数据联合建模； 图像 – 光谱 – 函数型三模态对齐应用于高光谱农产品检测。
